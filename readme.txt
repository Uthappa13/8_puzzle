# Project1_code

# Description
Project1_code makes use of queue to store the nodes to be visited and a set to store the nodes that have been visited. Once the goal node is reached, text files are created and the node path, node information and the nodes explored are written on to those files

Code blocks are provided with comments which explains what those code do.

# usage
Create a folder and paste the code file along with the text files in the folder.
The code can be run on VS code or any other code editor, input the initial state in a column wise fashion and the results will be written on to the text files.
Run the animate.py file from the same folder to visulaize the code results.
